# Markdown Viewer

A clean, real-time Markdown previewer built with React + TypeScript + Vite. Write on the left, see the rendered output on the right — instantly.

## Features

- **Live preview** — edits render as you type
- **Dark / light mode** — persists your preference
- **Toolbar shortcuts** — quick formatting buttons (bold, italics, headings, lists, code, links, images)
- **Syntax highlighting** — code blocks look great out of the box
- **Responsive layout** — works on desktop and mobile

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:5173` and start writing.

## Build for Production

```bash
npm run build
```

Output lands in `dist/`, ready to deploy anywhere (Vercel, Netlify, GitHub Pages, etc.).

## Tech Stack

- React 18 + TypeScript
- Vite for fast dev & builds
- `marked` for Markdown parsing
- `highlight.js` for code highlighting
- Plain CSS — no heavy UI libraries
